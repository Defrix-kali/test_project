# Лёгкий цветочный сайт

Запуск проекта (требуется .NET 7 SDK):

```bash
# Перейдите в корень репозитория и запустите
dotnet run
```

Сервер будет доступен на http://localhost:5000

Быстрая проверка: `GET /health` — вернёт `ok`.

Файлы фронтенда находятся в `wwwroot/` (index.html, admin.html, seller.html, client.html)
